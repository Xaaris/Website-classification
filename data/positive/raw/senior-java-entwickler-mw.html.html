Abwechslungsreiche und fordernde Tätigkeiten sind Ihre Leidenschaft? Seien Sie Teil eines vielfältigen Berufsfelds in einem interessanten Entwicklungsumfeld und bringen Sie Ihre Karriere auf ein neues Level. Bewerben Sie sich jetzt als …

Senior Java Entwickler (m/w)

Suchen Sie nach spannenden Herausforderungen, die Sie in einem motivierten Team von Softwareentwicklern lösen. Arbeiten Sie selbstständig an Projekten und übernehmen Sie Verantwortung, um innovative Ideen umzusetzen.

Ihre Aufgaben als Senior Java Entwickler

  • Aktive Entwicklung in Java EE und Open Source
  • Enge Zusammenarbeit mit den Algorithmen Teams
  • Designen und implementieren von Applikationen
  • Test und Dokumentation
  • Enger Austausch mit der Projektleitung
  • Einlernen von neuen Mitarbeitern
  • Stetige Weiterentwicklung der Software-Architektur

Ihr Profil als Senior Java Entwickler

  • Ein erfolgreich abgeschlossenes Studium im Bereich der IT, Mathematik, Physik oder ein vergleichbarer Abschluss
  • Mehrjährige praktische Berufserfahrung in der Software-Entwicklung
  • Fundierte Kenntnisse in der Programmierung mit Java sowie mit Open-Source-Frameworks wie JAVA EE, Hibernate, Spring, JBOSS & Tomcat, Swing, SQL, JMS
  • Erfahrung mit agilen Entwicklungsmethoden wie SCRUM sind von Vorteil
  • Ausgeprägte analytisch-konzeptionelle Fähigkeiten und Freude an innovativen Technologien
  • Sehr gute Kenntnisse in Deutsch und Englisch
  • Hohe Einsatzbereitschaft
  • Begeisterung an Verantwortungsübernahme
