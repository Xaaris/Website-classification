      Ich habe mein GMX Passwort vergessen

      Sollten Sie Ihr Passwort vergessen haben, können Sie es sich entweder in Ihrem Browser anzeigen lassen oder aber Ihr Passwort komplett zurücksetzen und ein neues vergeben. Auf https://passwort.gmx.net/ erhalten Sie schnell und einfach ein neues Passwort.

      Passwort vergessen

      Sie haben Ihr GMX Passwort vergessen oder können sich nicht mehr an die genaue Zeichenfolge erinnern.

      Passwort zurücksetzen
      1. Setzen Sie Ihr Passwort zurück, indem Sie die Seite https://passwort.gmx.net besuchen.
      2. Folgen Sie den Anweisungen auf Ihrem Bildschirm.
      Firefox: Passwort anzeigen
      1. Öffnen Sie das Menü mit einem Klick auf das Menü-Symbol.
      2. Klicken Sie auf Einstellungen.
      3. Klicken Sie auf Sicherheit und auf Gespeicherte Passwörter.... Es erscheint eine Liste mit gespeicherten Passwörtern.
      4. Um die Liste der Passwörter einzuschränken, geben Sie in das Sucheingabefeld "gmx" ein. Klicken Sie auf die Schaltfläche Passwörter anzeigen und bestätigen Sie mit einem Klick auf OK.
      Internet Explorer: Passwort anzeigen
      1. Bewegen Sie die Maus in die rechte obere Bildschirmecke und klicken Sie auf das Lupen-Symbol. Geben Sie Systemsteuerung in das Sucheingabefeld ein. Klicken Sie auf den entsprechenden Eintrag.
      2. Klicken Sie auf Benutzerkonten.
      3. Klicken Sie unter Anmeldeinformationsverwaltung auf Webanmeldeinformationen verwalten. Es erscheint eine Liste mit gespeicherten Passwörtern.
      4. Um die Liste der Passwörter einzuschränken, geben Sie in das Sucheingabefeld "gmx" ein. Klicken Sie auf den entsprechenden Eintrag und klicken Sie auf Einblenden. Geben Sie Ihr Windows-Benutzerkennwort ein und bestätigen Sie mit einem Klick auf OK.
      Chrome: Passwort anzeigen
      1. Öffnen Sie das Menü mit einem Klick auf das Menü-Symbol.
      2. Klicken Sie auf Einstellungen.
      3. Klicken Sie unten auf Erweiterte Einstellungen anzeigen....
      4. Klicken Sie im Bereich Passwörter und Formulare auf Passwörter verwalten. Es erscheint eine Liste mit gespeicherten Passwörtern.
      5. Um die Liste der Passwörter einzuschränken, geben Sie in das Sucheingabefeld "gmx" ein. Klicken Sie auf den entsprechenden Eintrag und klicken Sie auf Anzeigen.
      Safari: Passwort anzeigen
      1. Klicken Sie auf Safari und klicken Sie auf Einstellungen.
      2. Wechseln Sie zum Reiter Kennwörter (a). Es erscheint eine Liste mit gespeicherten Passwörtern.
      3. Um die Liste der Passwörter einzuschränken, geben Sie in das Sucheingabefeld "gmx" ein. (b). Setzen Sie das Häkchen neben Kennwörter für ausgewählte Websites einblenden (c). Geben Sie Ihr Mac-Nutzerpasswort ein und bestätigen Sie mit einem Klick auf OK.
      Thunderbird: Passwort anzeigen
      1. Um die Einstellungen zu öffnen, drücken Sie gleichzeitig die Tasten Alt + X Klicken Sie auf Einstellungen.
      2. Wechseln Sie zum Reiter Sicherheit und klicken Sie auf Passwörter. Klicken Sie auf die Schaltfläche Gespeicherte Passwörter.... Es erscheint eine Liste mit gespeicherten Passwörtern.
      3. Um die Liste der Passwörter einzuschränken, geben Sie in das Sucheingabefeld "gmx" ein. Klicken Sie auf Passwörter anzeigen und bestätigen Sie mit einem Klick auf Ja.
      Apple Mail: Passwort anzeigen
      1. Klicken Sie auf dem Mac-Schreibtisch in der Menüleiste unter Gehe zu auf Dienstprogramme.
      2. Klicken Sie doppelt auf die Schlüsselbundverwaltung. Ein neues Fenster öffnet sich.
      3. Um die Liste der Passwörter einzuschränken, geben Sie in das Sucheingabefeld "gmx" ein.
      4. Klicken Sie doppelt auf Ihre GMX E-Mail-Adresse. Die Einstellungen öffnen sich.
      5. Setzen Sie unten links das Häkchen neben Kennwort einblenden.
      6. Geben Sie Ihr Mac-Nutzerpasswort ein.
      7. Klicken Sie auf Erlauben.
