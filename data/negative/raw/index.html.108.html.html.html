DDS-CAD-Systemvoraussetzungen

Minimal

Betriebssystem
  • Windows 7 (SP1 oder höher) 32 oder 64 Bit (x86 oder x64)
  • Windows 8 (kein Windows RT) 32 oder 64 Bit (x86 oder x64)
  • Windows 8.1 (kein Windows RT) 32 oder 64 Bit (x86 oder x64)
  • Windows 10, 32 oder 64 Bit (x86 oder x64)
Betriebssystem für Dongleserver
  • Windows 7/8/8.1/10, 32 oder 64 Bit (x86 oder x64)
  • MS Windows Server 2003 oder höher
  • Diverse Linux-Distributionen
Prozessor (CPU)
  • Intel- oder AMD-Dual-Core-CPU mit 2,0 GHz oder mehr und SSE2
Arbeitsspeicher (RAM)
  • 4 GB
Bildschirm
  • Auflösung von 1280 x 1024 True Color
Grafikkarte (GPU)
  • OpenGL kompatible Grafikkarte, mindestens 1 GB RAM
Festplatte (HD)
  • 5 GB freier Speicherplatz
Optisches Laufwerk
  • DVD-Laufwerk (sofern die Installation von DVD erfolgen soll)
Maus
  • 3-Tasten-Radmaus
Tastatur (Keyboard)
  • MS-kompatibel
Weitere Systemvoraussetzungen
  • LAN (Gigabit oder 100 MBit/s) für Datenablage im Netzwerk und bei Betrieb über Netzwerkdongle
  • 1 x USB 2.0/3.0 bei Betrieb mit lokalem Dongle
  • Internetzugang für Updates

Empfohlen

Betriebssystem
  • Windows 7 Professional (SP1 oder höher), 64 Bit (x64)
  • Windows 8 Professional, 64 Bit (x64)
  • Windows 8.1 Professional, 64 Bit (x64)
  • Windows 10 Professional, 64 Bit (x64)
Betriebssystem für Dongleserver
  • MS Windows Server 2003 oder höher
Prozessor (CPU)
  • aktuelle Intel-Core-i5- oder -i7-CPU
  • vergleichbarer XEON-Prozessor
  • AMD FX ab Typ 6100
Arbeitsspeicher (RAM)
  • 8 GB
Bildschirm
  • Auflösung von 1920 x 1080 True Color
Grafikkarte (GPU)
  • Grafikkarte von Nvidia Quadro M1000 oder höher
  • Nvidia Quadro K1200 oder höher
Festplatte (HD)
  • 5 GB freier Speicherplatz, SSD-System empfohlen
Optisches Laufwerk
  • DVD-Laufwerk (sofern die Installation von DVD erfolgen soll)
Maus
  • Hochleistungs-Radmaus mit 3 oder mehr Tasten
Tastatur (Keyboard)
  • MS-kompatibel
Weitere Systemvoraussetzungen
  • Gigabit LAN für Datenablage im Netzwerk
  • Internetzugang für Updates
  • 1 x USB 2.0/3.0 für Dongle
