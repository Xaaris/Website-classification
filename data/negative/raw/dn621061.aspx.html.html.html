Eine alternative Position zu einer Angebotsanforderung hinzufügen [AX 2012]

Aktualisiert: October 9, 2014

Gilt für: Microsoft Dynamics AX 2012 R3

Dieses Feature ist nur verfügbar, wenn eProcurement in Microsoft Dynamics AX 2012 R3 installiert ist.

 Microsoft Dynamics AX 2012 R3 ermöglicht Kreditoren, Alternativen zu Ihren Angeboten hinzuzufügen. Als Einkaufsvertreter möchten Sie wahrscheinlich etwas über alternative Artikeln oder Dienstleistungen erfahren, falls es bessere Alternativen zu Ihrer ursprünglichen Anforderung gibt, oder der Artikel oder die Dienstleistung sich geändert hat. Ein Kreditor möchte möglicherweise eine breitere Palette oder neue Modelle anbieten. Sie müssen diese Funktionalität in den Einstellungen für die Angebotsanforderung (RFQ) aktivieren.
Hinweis

Wenn der Bestelltyp Bestellanforderung ist und alternative Positionen vorhanden sind, können Sie entweder die Original-Angebotsposition oder deren Alternative, aber nicht beides akzeptieren. Alternativen, die genehmigt wurden, werden in der Bestellanforderung angezeigt.

Wenn Sie darüber hinaus Gesamtkosten im Angebot prüfen, beachten Sie, dass die alternativen Positionen nicht in die Summe einbezogen werden sollen.

Alternative Positionen in einer Angebotsanforderung zulassen

Verwenden Sie dieses Verfahren, wenn Kreditoren in der Lage sein sollen, Ersatz oder Alternativen in ihren Angeboten einzubeziehen.

  1. Klicken Sie auf Beschaffung > Häufig > Angebotsanforderungen > Alle Angebotsanforderungen.

  2. Öffnen Sie die Angebotsanforderung, mit der Sie arbeiten möchten.

  3. Aktivieren Sie in der Angebotsanforderungskopf unter Informationen zur Angebotsanforderung das Kontrollkästchen Alternativen für Antwortpositionen zulassen.

Alternative Positionen in einer Antwort auf Angebotsanforderung erstellen

Wenn die Funktionen für das Hinzufügen von Alternativen aktiviert wurden, können Sie dieses Verfahren verwenden, wenn Sie ein Einkaufsvertreter sind und alternative Positionen hinzufügen möchten, wenn Sie eine Antwort auf Angebotsanforderung im Voraus definieren. Der Status der Angebotsanforderung muss Versendet lauten. Nur nicht im Katalog enthaltene Artikel (Kategorieartikel) können Alternativen haben. Weitere Informationen finden Sie unter Erstellen einer Antwort auf eine Angebotsanforderung.

Hinweis

Wenn Sie ein Kreditor sind und alternative Positionen hinzufügen möchten, wenn Sie auf eine Angebotsanforderung im Kreditorenportal auf Enterprise Portal für Microsoft Dynamics AX antworten, vgl. Anzeigen und Beantworten von einer Angebotsanforderung. Wenn Sie ein Kunde aus dem Sektor öffentliche Hand sind, vergleichen Sie Auf eine Angebotsanforderung hin ein Angebot machen.

  1. Klicken Sie auf Beschaffung > Häufig > Angebotsanforderungen > Antworten auf Angebotsanforderung.

  2. Öffnen Sie auf der Listenseite Antworten auf Angebotsanforderung die Antwort auf eine Angebotsanforderung, die Sie verwenden möchten. Klicken Sie im Aktionsbereich auf der Registerkarte Antworten auf Bearbeiten.

  3. Wählen Sie im Inforegister Bestellanforderungspositionen die Kategorieposition aus, unter der die Alternative erscheinen soll.

  4. Klicken Sie auf Alternative hinzufügen.

  5. Hier können Sie Informationen zum Produkt, beispielsweise Artikelnummer oder Beschaffungskategorie, Menge, Preis und Rabatt, eingeben.
