    Wiederherstellen gelöschter Elemente oder E-Mails in Outlook Online

    Gilt für: Outlook Web App für Office 365 Outlook Web App für Office 365 Small Business Outlook Web App Mehr... Weniger

    Dieser Artikel bezieht sich auf Outlook Web App. Verwenden Sie etwas anderes?

    Versuchen Sie es mit Outlook.com | Outlook im Web | Outlook 2013 oder Outlook 2016 | Outlook 2010

    Wenn Sie versehentlich ein Element aus Ihrem Outlook Web App-Postfach löschen (der Webbrowserversion von Outlook), können Sie es häufig wiederherstellen. Der erste Ort, an dem Sie nach einem gelöschten Element wie einer E-Mail-Nachricht, einem Kalendertermin, einem Kontakt oder einer Aufgabe suchen sollte, ist der Order "Gelöschte Elemente" in Ihrem Postfach. Wenn Sie es dort nicht finden, sollten Sie als Nächstes im Ordner "Wiederherstellbare Elemente" nachschauen, auf den Sie mit dem Tool Gelöschte Elemente wiederherstellen zugreifen können.

    Müssen Sie einen gelöschten E-Mail-Ordner in Outlook Web App wiederherstellen?    Sie können einen gelöschten Ordner (mit allen darin befindlichen Nachrichten) wiederherstellen, sofern er sich noch im Ordner "Gelöschte Elemente" befindet – So wird's gemacht. Ein Ordner, der dauerhaft gelöscht wurde, kann leider nicht mehr wiederhergestellt werden. Sie können mit den Schritten in diesem Artikel jedoch die Nachrichten aus dem Ordner "Wiederherstellbare Elemente" wiederherstellen, die sich in einem Ordner befunden haben, als dieser dauerhaft gelöscht wurde.

    Wenn Sie eine E-Mail-Nachricht, einen Kontakt, ein Kalenderelement oder eine Aufgabe löschen, wird das Element in den Ordner "Gelöschte Elemente" in Ihrem Postfach verschoben. Wenn Sie versuchen, eine gelöschte Nachricht zu finden, ist der Ordner "Gelöschte Elemente" der erste Ort, an dem Sie suchen sollten. Wenn sich die Nachricht hier befindet, können Sie sie wie folgt wiederherstellen:

    1. Melden Sie sich bei Ihrem Konto an. Weitere Informationen finden Sie unter Anmelden bei Outlook Web App.

    2. Klicken Sie in der Liste der E-Mail Ordner auf den Ordner Gelöschte Elemente.

    3. Wenn Sie die gesuchte Nachricht gefunden haben, führen Sie einen der folgenden Schritte aus, um sie wiederherzustellen:

      • Wenn eine Nachricht in Ihrem Posteingang wiederhergestellt werden soll, klicken Sie mit der rechten Maustaste darauf, und klicken Sie dann auf Verschieben > Posteingang.

      • Wenn eine Nachricht in einem anderen Posteingang wiederhergestellt werden soll, klicken Sie mit der rechten Maustaste darauf, und klicken Sie dann auf Verschieben > Mehr. Wählen Sie einen Ordnerspeicherort aus, und klicken Sie anschließend auf Verschieben.

    4. Wenn Sie andere Elementtypen aus dem Ordner "Gelöschte Elemente" wiederherstellen möchten, führen Sie eine der folgenden Aktionen aus:

      • Zum Wiederherstellen eines Kontakts klicken Sie mit der rechten Maustaste darauf, und klicken Sie dann auf In Kontakte verschieben.

      • Zum Wiederherstellen eines Kalenderelements klicken Sie mit der rechten Maustaste darauf, und klicken Sie dann auf In Kalender verschieben.

      • Zum Wiederherstellen einer Aufgabe klicken Sie mit der rechten Maustaste darauf, und klicken Sie dann auf In Aufgaben verschieben.

    Tipps zum Wiederherstellen von Elementen aus dem Ordner "Gelöschte Elemente"

    • Wenn Sie einen Ordner löschen, wird dieser in den Ordner "Gelöschte Elemente" verschoben und dort als Unterordner angezeigt. Um den Ordner (und alle hierin befindlichen Elemente) wiederherzustellen, klicken Sie mit der rechten Maustaste auf den Ordner, dann auf Verschieben, und wählen Sie einen Ordner aus, in den er verschoben werden soll.

    • Zum die Suche nach Elementen im Ordner "Gelöschte Elemente" zu vereinfachen, können Sie nach Typ sortieren. Damit werden die Postfachelemente nach Typen gruppiert.

    • Möglicherweise hat Ihr Administrator eine Richtlinie erstellt, mit der gelöschte Elemente nach einer bestimmten Anzahl Tagen aus dem Ordner "Gelöschte Elemente" entfernt werden. Ebenso wie ein von Ihnen gelöschtes Element werden auch von einer Richtlinie gelöschte Elemente in den Ordner "Wiederherstellbare Elemente" verschoben. Wenn Sie das gewünschte Element also im Ordner "Gelöschte Elemente" nicht finden können, schauen Sie im Ordner "Wiederherstellbare Elemente" nach. Im nächsten Abschnitt wird erläutert, wie Sie vorgehen müssen.

    Seitenanfang

    Wiederherstellen eines Elements, das sich nicht mehr im Ordner "Gelöschte Elemente" befindet

    Wenn Sie das gewünschte Element im Ordner "Gelöschte Elemente" nicht mehr finden können, sollten Sie als Nächstes im Ordner "Wiederherstellbare Elemente" nachschauen. Dies ist ein verborgener Ordner und der Speicherort, an den Elemente verschoben werden, wenn Sie eine der folgenden Aktionen ausführen:

    • Löschen eines Elements aus dem Ordner "Gelöschte Elemente"

    • Leeren des Ordners "Gelöschte Elemente"

    • Dauerhaftes Löschen eines Elements durch Markieren und Drücken von Umschalt+ENTF.

    So können Sie Elemente aus dem Ordner "Wiederherstellbare Elemente" wiederherstellen

    1. Melden Sie sich bei Ihrem Konto an. Weitere Informationen finden Sie unter Anmelden bei Outlook Web App.

    2. Klicken Sie in der Liste der E-Mail-Ordner mit der rechten Maustaste auf Gelöschte Elemente, und klicken Sie dann auf Gelöschte Elemente wiederherstellen.

    3. Falls erforderlich, verwenden Sie das Suchfeld, um das Element zu finden, das Sie wiederherstellen möchten.

    4. Wenn Sie das Element gefunden haben, markieren Sie es, und klicken Sie dann auf Wiederherstellen.

    Tipp : Sie können E-Mail-Nachrichten, Kontakte, Kalenderelemente und Aufgaben aus dem Ordner "Wiederherstellbare Elemente" wiederherstellen.

    Wo werden die wiederhergestellten Elemente gespeichert?    Das Element wird an den Standardspeicherort für diesen Elementtyp verschoben. Dies bedeutet:

    • Nachrichten werden im Posteingang abgelegt.

    • Kalenderelemente werden in Ihrem Kalender abgelegt.

    • Kontakte werden in Ihrem Kontakteordner abgelegt.

    • Aufgaben werden in Ihrem Aufgabenordner abgelegt.

    Seitenanfang

    Tipps zum Wiederherstellen von Elementen aus dem Ordner "Wiederherstellbare Elemente"

    • Wenn Sie Outlook.com (vormals Hotmail.com) verwenden, lesen Sie die folgenden Artikel zum Wiederherstellen eine gelöschten E-Mail-Nachricht oder eines gelöschten Kontakts.

      • Wiederherstellen von gelöschten E-Mail-Nachrichten in Outlook.com

      • Wiederherstellen von gelöschten Kontakten in Outlook.com

    • Das Löschdatum eines Elements im Ordner "Wiederherstellbare Elemente" gibt an, wann das Elemente ursprünglich gelöscht und in den Ordner "Gelöschte Elemente" verschoben wurde. Dies ist nicht das Datum, zu dem das Elemente dauerhaft gelöscht und in den Ordner "Wiederherstellbare Elemente" verschoben wurde.

    • Wenn der Ordner "Wiederherstellbare Elemente" eine große Menge Elemente enthält, kann es schwierig sein, das Gewünschte zu finden, da die Elemente keine Symbole haben und alle sehr ähnlich aussehen. Hier sind einige Tipps:

      • Wenn Sie nach einem Kontakt suchen, suchen Sie nach dem Namen der Person. Kontakte haben zudem auch kein Feld "Betreff".

      • Wenn Sie nach einem Kalendertermin suchen, suchen Sie nach einem Element, das weder den Namen einer Person noch eine Betreffzeile aufweist.

      • Wenn Sie nach einem Kontakt, einem Kalendertermin oder einer Aufgabe suchen, suchen Sie nach dem Text Keine Vorschau verfügbar.

    • Zum Wiederherstellen mehrerer Elemente klicken Sie auf die Kontrollkästchen neben den einzelnen Elementen, und klicken Sie dann auf Wiederherstellen. Das Kontrollkästchen wird erst angezeigt, wenn Sie darauf zeigen.

      Sie können mehrere aufeinander folgende Elemente auch wiederherstellen, indem Sie das Kontrollkästchen des ersten Elements aktivieren, dann die UMSCHALTTASTE gedrückt halten und auf das Kontrollkästchen des letzten Elements klicken. Wenn alle Elemente markiert sind, klicken Sie auf Wiederherstellen.

    • Sie können Elemente, die im Fenster Gelöschte Elemente wiederherstellen angezeigt werden, auch dauerhaft löschen. Markieren Sie einfach das Element, und klicken Sie dann auf Löschen. Aber seien Sie vorsichtig: Wenn Sie ein Element dauerhaft gelöscht haben, können Sie es über "Gelöschte Elemente wiederherstellen" nicht mehr wiederherstellen. Sie müssen den Administrator fragen, ob er in der Lage ist, ein auf diese Weise dauerhaft gelöschtes Element wiederherzustellen.

    • Wenn Sie in einer Organisation arbeiten, hat der Administrator möglicherweise festgelegt, wie lange Elemente im Ordner "Wiederherstellbare Elemente" für eine Wiederherstellung verfügbar sind. So gibt es möglicherweise eine Richtlinie, mit der alles, was sich seit 30 Tagen in Ihrem Ordner "Gelöschte Elemente" befindet, gelöscht wird, und eine weitere Richtlinie, die dafür sorgt, dass Sie Elemente im Ordner "Wiederherstellbare Elemente" noch weitere 14 Tage lang wiederherstellen können. Und auch nach diesen 14 Tagen ist der Administrator möglicherweise in der Lage, ein Element wiederherzustellen. Wenn Sie ein Element im Ordner "Wiederherstellbare Elemente" nicht finden können, fragen Sie den Administrator, ob er das Element für Sie wiederherstellen kann.
