FAQ (Frequently Asked Questions)

Absatzvorlage, um Fragen und Antworten in einer FAQ-Liste anzuzeigen

  • Absatzvorlagen
    • Erweitert
      • FAQ

Diese Absatzvorlage erstellt eine Frage-/Antwortliste (FAQ).

In der erstellten Liste können frei formulierte Fragen und die dazugehörigen Antworten eingetragen werden. Zum Einen können die Antworten zu den jeweiligen Fragen über die Funktion „++ Zeige alle“ bzw. „-- Verstecke alle“ gleichzeitig ein- bzw. ausgeblendet werden. Zum Anderen kann nur eine Antwort auf eine spezielle Frage angezeigt werden, indem auf das + Symbol am Anfang der jeweiligen Frage geklickt wird.

Ein Beispiel für eine mögliche Anwendung des Absatzes finden Sie hier (nur mit TU-Login)

Karteireiter „FAQ“

„Überschrift“

Geben Sie optional eine Überschrift für den Absatz ein. Sie wird größer und in der Hervorhebung Ihrer Webauftrittsfarbe dargestellt.

Mit einem Klick auf dieses Symbol können Sie einen neuen Eintrag in Ihrer FAQ Liste erzeugen. Nach dem Klick wird das Fenster „FAQ: Eintrag in FAQ“ (siehe unten) geöffnet, welches im Folgenden näher beschrieben wird.

Mit einem Klick auf dieses Symbol können Sie Einträge aus der FAQ Liste entfernen, dazu wählen Sie den entsprechenden Eintrag aus der Liste aus und klicken anschließend auf dieses Symbol.

Sie können Einträge in der Liste beliebig nach oben oder unten verschieben. Dazu wählen Sie den zu verschiebenden Eintrag aus und klicken anschließend auf die entprechenden Pfleiltasten.

Mit einem Klick auf dieses Symbol können Sie einen Eintrag in der FAQ Liste bearbeiten, z.B. um Veränderungen oder Ergänzungen vorzunehmen. Es öffnet sich wieder das Fenster „FAQ: Eintrag in FAQ“ (siehe unten) mit Ihren schon eingegeben Daten.

Fenster FAQ: Eintrag in FAQ

„Zwischenüberschrift“

Hier kann optional eine Zwischenüberschrift angegeben werden. Damit können die Fragen & Antworten optisch gruppiert werden. Die Zwischenüberschrift wird etwas kleiner als eine normale Überschrift dargestellt.

„Frage“

In diesem Textfeld können Sie Ihre Frage eingeben.

„Antwort“

In diesem Textfeld können Sie die Antwort zu der oben eingegeben Frage eingeben.

„Bilder“

Hier können Sie, ähnlich wie bei der Erstellung der FAQ Liste, Bilder zu Ihren Einträgen hinzufügen. Wählen Sie dazu ein Bild aus dem Medienbereich aus. Das Bild wird neben dem Antworttext angezeigt.

Karteireiter „Anzeigeoptionen“

„'Zeige alle' + 'Verstecke alle' anzeigen?“

Wählen Sie diese Option aus, um die zusätzlichen Symbole „++ Zeige alle“ bzw. „-- Verstecke alle“ oberhalb der FAQ Liste anzuzeigen. Diese Funktion dient dazu alle Antworten zu den Fragen anzuzeigen bzw. zu verstecken (siehe oben).

„Größe der Bilder?“

Wählen Sie aus, in welcher Art die Bilder (optional) neben dem Antworttext angezeigt werden sollen. Sie können hierbei zwischen halbspaltig oder viertelspaltig auswählen.

„Ebene der Überschrift?“

Wählen Sie aus, in welcher Überschriftsgröße die Überschrift dargestellt werden sollen. Überschriften der Ebene 1 sind Hauptüberschriften, Überschriften der Ebene 2 sind Nachüberschriften und etwas kleiner als Hauptüberschriften.

Karteireiter „Zugriff“ (Inhaltsbereich)

„Zugriff nur für Besitzer einer TU-ID“

Wird diese Option ausgewählt, so ist der Absatz nur sichtbar, wenn der Nutzer mit einer TU-ID auf der Website angemeldet ist. Ansonsten ist der Absatz nicht sichtbar.

„Zugriff nur für diese globalen Benutzergruppen“

Wird diese Option ausgewählt, so ist der Absatz nur sichtbar, wenn der Nutzer mit einer TU-ID auf der Website angemeldet ist.

Gehört die TU-ID einem Mitarbeiter, so ist der Absatz sichtbar, wenn hier „Mitarbeiter“ ausgewählt wurde. Gehört die TU-ID einer/m Studierenden, so ist der Absatz sichtbar, wenn „Studierende“ ausgewählt wurde. Gehört die TU-ID zu einer externen Person mit TU-ID so ist der Absatz sichtbar, wenn „Externe mit TU-ID“ ausgewählt wurde.

Für Nutzer, die sich nicht mit einer TU-ID anmelden, ist der Absatz nicht sichtbar.

„Zugriff einschränken auf Liste von TU-IDs“

Wird der Bereich „Zugriff einschränken auf Liste von TU-IDs“ ausgewählt, so können nur jene Personen, deren TU-ID in der Liste eingetragen wird, auf den entsprechenden Bereich zugreifen.

„Auf geschützten Absatz hinweisen, wenn Benutzer nicht angemeldet ist?“

Wird diese Option gewählt, so erhalten nicht angemeldete Nutzer an Stelle des Absatzes einen Hinweistext, dass der Absatz nur für angemeldete Benutzer mit TU-ID sichtbar ist.
