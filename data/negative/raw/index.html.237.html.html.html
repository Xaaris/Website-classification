Facebook: Neues Beitragsformat “Job” auf Facebook Seiten!

  • Share / Save
  • Home
  • Facebook Seiten / Pages
  • Facebook: Neues Beitragsformat “Job” auf Facebook Seiten!
  • 17.08.2016
  • Thomas Hutter
  • Facebook Seiten / Pages, HR 2.0

Still und heimlich hat Facebook ein neues Beitragsformat mit dem Namen “Job” für Facebook Seiten lanciert. Unternehmen können nun neu Stellenanzeigen direkt als Beitragsart publizieren. Publizierte Beiträge werden automatisch in einem neuen Tab “Jobs” angezeigt.

Beitragsart “Job” auf Facebook Seiten

Begrüssungsmeldung im Tab “Jobs”

 

Klickt man auf “get startet” im Tab “Jobs” oder in der Publisher-Box auf “Q&A, Offer, Job+”, also der “Allerlei”-Rubrik, findet man auf Facebook Seiten (US-Englische Spracheinstellung) eine neue Beitragsart “Job”.

Neue Beitragsart “Job” für Facebook Seiten

Nach dem Klick auf “Job” öffnet sich ein Publishing-Dialog in einem neuen Fenster:

Publisher-Box für Stellenanzeigen

 

Facebook stellt für die Publikation folgende Felder zur Verfügung:

  • Einleitung / Introduction
    ein optionales Feld mit einer maximalen Länge von 90 Zeichen.
  • Photo
    mit einer Upload-Möglichkeit eines Bildes, wahlweise kann auch das Titelbild der Facebook Seite verwendet werden.
  • Stellentitel / Job Title
    ein Feld mit einer maximalen Länge von 50 Ziechen
  • Ort / Location
    bei Ort kann ein bestehender Facebook Ort ausgewählt werden. Eine Freitexteingabe ist nicht möglich.
  • Lohn / Salery
    ein optionales Feld mit der Angabe des Lohns und den Auswahlmöglichkeiten “pro Stunde / per hour”, “pro Tag / per day”, “pro Woche / per week”, “pro Monat / per month” oder “pro Jahr / per year”. Leider steht aktuell nur US-$ als Währung zur Verfügung.
  • Job Type
    Die Art des Jobs, zur Auswahl stehen “Vollzeit / Full-time”, “Teilzeit / Part-time”, “Praktikum / Internship” oder “Vertrag / Contract”
  • Details
    Für die Beschreibung stehen unter “Details” 5000 Zeichen zur Verfügung.

Tab “Jobs”

Hat man einen Job-Beitrag publiziert, wird dieser im News Feed verbreitet und gleichzeitig im Tab “Jobs” angezeigt.

Klickt man im Tab “Jobs” auf “Apply”, wird der Job im Vollformat dargestellt und alle Details sichtbar zusätzlich zu den eingegebenen Informationen zieht Facebook auch die Postadresse aus der dem hinterlegten Ort.

Stellenanzeige im Vollformat

Klickt man anschliessend wieder auf “Apply”, öffnet sich ein Dialogfenster für die Bewerbung. Neben dem Namen, Telefonnummer, Ort und E-Mail-Adresse füllt Facebook automatisch die im Profil hinterlegten Job-Positionen und Ausbildungen hinzu, welche für das Einreichen der Bewerbung wieder entfernt werden können.

Bewerbungsformular mit vorausgefüllten Daten.

 

Leider ist das Hinzufügen von Dokumenten nicht möglich.

Wo gehen die Bewerbungen hin?

Ja, wo gehen die Bewerbungen hin? Abgesendete Formulare werden als Nachricht an die Facebook Seite gesendet und stehen im Messenger zur Verfügung.

Mein Gefühl sagt mir, dass dies aus Sicht des Datenschutz wahrscheinlich ein Problem in Deutschland darstellen wird ….

 

Verfügbarkeit

Die Funktion steht aktuell nur auf Facebook Seiten mit der Spracheinstellung “US-English” zur Verfügung, allerdings konnte ich bei einer kurzen Suche feststellen, dass die neue Funktion nicht auf allen Seiten zur Verfügung steht – es ist daher davon auszugehen, dass die Funktion sich im Rollout befindet.
