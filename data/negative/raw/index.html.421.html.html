Bewerbungsfrist für Studienplatzbewerbung

Als Studienanfänger/in (d.h. eine Bewerbung in das erste Fachsemester) können Sie sich an der HafenCity Universität nur zum Wintersemester bewerben. Eine Bewerbung auf ein höheres Fachsemester kann sowohl zum Wintersemester (3. und 5. Fachsemester) als auch zum Sommersemester (2. oder 4. Fachsemester) erfolgen.

Bewerbungszeitraum für das Wintersemester:*
01. Juni - 15. Juli

*Bitte beachten Sie zum Wintersemester die abweichenden Bewerbungsfristen für die Studiengänge Master Architektur und Master REAP. Für diese beiden Studiengänge können Sie sich vom 01.Juni - 01.Juli bewerben.

Bewerbungszeitraum für das Sommersemester:
01. Dezember - 15. Januar
