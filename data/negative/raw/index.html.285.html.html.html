      please post the output of
      cat /etc/cron.d/openmediavault-userdefined
      it should look something like this:

      Source Code

      1. ~$ cat /etc/cron.d/openmediavault-userdefined
      2. SHELL=/bin/sh
      3. PATH=/usr/local/sbin:/usr/local/bin:/sbin:/bin:/usr/sbin:/usr/bin
      4. # m h dom mon dow user command
      5. */30 * * * * root /var/lib/openmediavault/cron.d/userdefined-8a0760ab-d82d-4231-8c99-4e0e0d00f141 >/dev/null 2>&1

      another thing you can try is to execute the command from your output. In this example would be:

      Source Code

      1. /var/lib/openmediavault/cron.d/userdefined-8a0760ab-d82d-4231-8c99-4e0e0d00f141

      and see, if it does, what you want.
